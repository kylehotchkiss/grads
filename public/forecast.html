Forecast

Visualize forecast data for any country in the world. Based off of NOAA's GFS models, or RAP if you choose. Given that this is dynamically generating GeoJSON for Mapbox, it's not terribly efficent but I hope to eventually find a way to run these via Mapbox native and generate some nicer gifs on an automated basis.
